{{% alert color="primary" %}}

Aspose.CAD를 JasperReports에 설치하려면 시스템이 다음 요구 사항을 충족해야 합니다:

- Java 2 Platform Standard Edition 1.6 이상.
- JasperReports 6.3.0 이상.
- JasperReports Server 6.3.0 이상.

{{% /alert %}}
