## **아직 지원되지 않음**

- 그림자, 조명, 재료, 추가 카메라 등.
- 같은 도면에서 3D 엔티티와 자연스럽게 평면인 엔티티가 혼합된 경우 지원되지 않음.
- 교차된 엔티티, 결합된 엔티티 등과 같은 엔티티 구축 지원되지 않음.
