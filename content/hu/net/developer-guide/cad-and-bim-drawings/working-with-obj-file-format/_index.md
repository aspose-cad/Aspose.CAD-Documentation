{{% alert color="primary" %}}

Az Aspose.CAD mostantól támogatja az OBJ fájlformátumot. Az OBJ fájlformátum egy 3D geometria, amely tartalmaz textúratérképeket, 3D koordinátákat, poligonális felületeket és egyéb objektuminformációkat.

{{% /alert %}}
