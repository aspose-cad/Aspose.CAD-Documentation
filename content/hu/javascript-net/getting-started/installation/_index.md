## **Aspose.CAD telepítése JavaScript-hez npm-en keresztül**

Ahhoz, hogy az Aspose.CAD-ot JavaScript-ből használd az alkalmazásodban, telepítsd az aspose.cad-ot a [NPM](https://www.npmjs.com/@aspose-cad/) segítségével a projektedben. Ezt követően programozottan hozzáférhetsz az exportálókhoz.

**JavaScript**

{{< highlight python >}}
npm install aspose-cad
{{< /highlight >}}
