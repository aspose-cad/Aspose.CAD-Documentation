{{% alert color="primary" %}}

Az Aspose.CAD most már támogatja az OBJ fájlformátumot. Az OBJ fájlformátum egy 3D geometria, amely textúratérképeket, 3D koordinátákat, poligonális felületeket és egyéb objektuminformációkat tartalmaz.

{{% /alert %}}
