{{% alert color="primary" %}}

لتثبيت Aspose.CAD لـ JasperReports على نظامك، يجب أن يلبي نظامك المتطلبات التالية:

- Java 2 Platform Standard Edition 1.6 أو أعلى.
- JasperReports 6.3.0 أو أعلى.
- JasperReports Server 6.3.0 أو أعلى

{{% /alert %}}
