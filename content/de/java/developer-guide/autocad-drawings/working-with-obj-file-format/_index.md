Aspose.CAD unterstützt jetzt das OBJ-Dateiformat. Das OBJ-Dateiformat ist eine 3D-Geometrie, die Texturenmappen, 3D-Koordinaten, polygonale Flächen und andere Objektinformationen enthält.

{{% /alert %}}
