## **Ambientazioni di Sviluppo**

Puoi utilizzare Aspose.CAD per .NET per sviluppare applicazioni in qualsiasi ambiente di sviluppo che supporti la piattaforma .NET, ma i seguenti ambienti sono esplicitamente supportati:

- Microsoft Visual Studio 2010
- Microsoft Visual Studio 2012
- Microsoft Visual Studio 2013
- Microsoft Visual Studio 2015
- Microsoft Visual Studio 2017
- Microsoft Visual Studio 2019
- MonoDevelop 2.4 e versioni successive
