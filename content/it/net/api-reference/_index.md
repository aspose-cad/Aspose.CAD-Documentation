Aspose.CAD per .NET è un'API autonoma per l'elaborazione di file CAD e BIM adatta per .Net Core, Xamarin per Android e iOS.
Fornisce la funzionalità di convertire file DXF, DWG, DGN, DWF, DWFX, IFC, STL, DWT, IGES, PLT, CF2, OBJ, HPGL, IGS, PCL, FBX, SVG in PDF di alta qualità e altri file immagine vettoriali e raster come: BMP, GIF, JPG, JPEG, JP2, PNG, TIFF, PSD, DICOM, WEBP, WMF, EMF.
Gli sviluppatori possono selezionare e convertire layout e livelli specifici dai file AutoCAD, nonché monitorare l'intero processo di conversione del file con facilitá.
L'API Aspose.CAD per .NET è multi-thread safe e pronta per l'elaborazione parallela in massa con supporto per la cancellazione.

Aspose.CAD per .NET è completamente scritto in C#, è conforme a CLS e contiene solo codice gestito sicuro.

L'ultimo Riferimento API può essere trovato su [Riferimento API Aspose.CAD per .NET](https://reference.aspose.com/cad/net/).
