## **Benvenuto in Aspose.CAD per JavaScript**

![Aspose.CAD for JavaScript Product Logo](/cad/_assets/home_5.png)

Aspose.CAD per JavaScript Ã¨ un'API nativa di conversione CAD senza dipendenze software. Consente agli sviluppatori di convertire file DXF, DWG, DWT, DGN, IFC, DWF, DWFX, STL, IGES(IGS), CF2, Collada(DAE), PLT, OBJ, SVG, DXB, FBX, U3D, 3DS, STP in formati immagine raster e PDF.
