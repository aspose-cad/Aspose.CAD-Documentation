## **לא נתמך עדיין**

- צללים, אורות, חומרים, מצלמות נוספות, וכו'.
- אין תמיכה לגידול טבעי מעורב עם גידולים תלת מימדיים באותו רישום.
- אין תמיכה ליחידות בנייה כמו יחידות חופפות, יחידות מאוחדות וכו'.
