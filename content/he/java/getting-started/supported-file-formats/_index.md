## **עדיין לא נתמך**

- צללים, אורות, חומרים, מצלמות נוספות וכו'.
- אין תמיכה ליצור ישויות שטוחות מעורבות עם ישויות תלת ממדיות באותו תרשים.
- אין תמיכה בבניית ישויות כמו ישויות החופפות, ישויות מאוחדות וכו'.
