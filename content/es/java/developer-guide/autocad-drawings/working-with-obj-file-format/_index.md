{{% alert color="primary" %}}

Aspose.CAD ahora admite el formato de archivo OBJ. El formato de archivo OBJ es una geometría 3D que contiene mapas de textura, coordenadas 3D, caras poligonales y otra información del objeto.

{{% /alert %}}
