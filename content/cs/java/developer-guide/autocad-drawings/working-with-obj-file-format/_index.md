{{% alert color="primary" %}}

Aspose.CAD nyní podporuje formát souboru OBJ. Formát souboru OBJ je 3D geometrie obsahující texturové mapy, 3D souřadnice, polygonální plochy a další informace o objektu.

{{% /alert %}}
