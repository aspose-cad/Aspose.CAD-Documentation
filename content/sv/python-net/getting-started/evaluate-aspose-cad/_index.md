{{% alert color="primary" %}}

Du kan enkelt ladda ner Aspose.CAD för Python för utvärdering från [nedladdningssidan](https://downloads.aspose.com/cad/python-net). Nedladdningen för utvärdering är densamma som den köpta nedladdningen. Utvärderingsversionen blir helt enkelt licensierad efter [att ha tillämpat en licens](/cad/python-net/licensing/).

{{% /alert %}}

Utvärderingsversionen av Aspose.CAD för Python (utan specificerad licens) ger produktfunktionalitet till viss del men är inte heltäckande. En begränsning är att den alltid injicerar utvärderingsvattenstämpeln på utdatafiler.

**Figur:** Utvärderingsvarning

Om du vill testa Aspose.CAD för Python utan begränsningar i utvärderingsversionen kan du [begära en 30-dagars temporär licens](https://purchase.aspose.com/temporary-license).
