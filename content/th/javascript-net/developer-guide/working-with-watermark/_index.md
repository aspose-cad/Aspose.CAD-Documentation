## **เพิ่มลายน้ำใน DGN**

บางครั้งคุณอาจต้องการเพิ่มลายน้ำให้กับแบบวาดเพื่อระบุวัตถุประสงค์ของแบบวาดหรือต้นกำเนิดของมัน Aspose.CAD สำหรับ Python ช่วยให้คุณสามารถเพิ่มลายน้ำในไฟล์ DGN ได้
