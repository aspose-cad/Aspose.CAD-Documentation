Aspose.CAD ข ahora สนับสนุนรูปแบบไฟล์ OBJ รูปแบบไฟล์ OBJ เป็นเรขาคณิต 3 มิติที่มีแผนที่พื้นผิว, พิกัด 3 มิติ, หน้าพอลิกอน, และข้อมูลวัตถุอื่น ๆ
