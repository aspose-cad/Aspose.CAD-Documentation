Aspose.CAD ขณะนี้รองรับรูปแบบไฟล์ OBJ รูปแบบไฟล์ OBJ เป็นรูปทรง 3 มิติที่มีแผนที่พื้นผิว, พิกัด 3 มิติ, หน้าพอลิกอน, และข้อมูลวัตถุอื่นๆ

{{% /alert %}}
