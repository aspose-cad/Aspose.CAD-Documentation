## **Formats de Fichier Supportés**

Le tableau suivant indique les formats de fichier que Aspose.CAD pour Python peut charger et enregistrer.

|[DGN](https://docs.fileformat.com/cad/dgn/)|Fichier de Design MicroStation|{{< emoticons/tick >}}| |- Solides 3D (Conique, Sphère, Torus, Cylindre, Boîte, Coin)<br />- Surfaces, Maillages|
