Aspose.CAD pour .NET est une API autonome de traitement de fichiers CAD et BIM adaptée pour .Net Core, Xamarin pour Android et iOS.
Elle offre la fonctionnalité de convertir des fichiers DXF, DWG, DGN, DWF, DWFX, IFC, STL, DWT, IGES, PLT, CF2, OBJ, HPGL, IGS, PCL, FBX, SVG en PDF de haute qualité et d'autres fichiers image vectoriels et matriciels tels que : BMP, GIF, JPG, JPEG, JP2, PNG, TIFF, PSD, DICOM, WEBP, WMF, EMF.
Les développeurs peuvent sélectionner et convertir des mises en page et des calques spécifiques des fichiers AutoCAD ainsi que suivre l'ensemble du processus de conversion de fichier avec facilité.
L'API Aspose.CAD pour .NET est sécurisée pour le multi-threading et prête pour un traitement parallèle en masse avec support d'annulation.

Aspose.CAD pour .NET est entièrement écrit en C#, conforme à la CLS, et contient uniquement du code géré sûr.

La dernière référence de l'API peut être trouvée à [Référence de l'API Aspose.CAD pour .NET](https://reference.aspose.com/cad/net/).
