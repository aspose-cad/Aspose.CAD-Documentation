## **Поддържани файлови формати**

|[DGN](https://docs.fileformat.com/cad/dgn/)|MicroStation Design File|{{< emoticons/tick >}}| |- 3D тела (конични, сферични, торически, цилиндрични, кутии, клинове)<br />- Повърхности, мрежи.|
