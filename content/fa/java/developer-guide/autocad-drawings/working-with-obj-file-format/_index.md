{{% alert color="primary" %}}

Aspose.CAD حالا از فرمت فایل OBJ پشتیبانی می‌کند. فرمت فایل OBJ یک هندسه سه‌بعدی است که شامل نقشه‌های بافت، مختصات سه‌بعدی، صورت‌های چندضلعی و سایر اطلاعات شیء می‌باشد.

{{% /alert %}}
