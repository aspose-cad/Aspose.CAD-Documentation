{{% alert color="primary" %}}

Aspose.CAD اکنون از فرمت فایل OBJ پشتیبانی می‌کند. فرمت فایل OBJ یک هندسه 3 بعدی است که شامل نقشه‌های بافت، مختصات 3 بعدی، چهره‌های چندضلعی و سایر اطلاعات اشیاء است.

{{% /alert %}}
