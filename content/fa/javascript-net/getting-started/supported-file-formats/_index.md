## **هنوز پشتیبانی نشده**

- سایه‌ها، نورها، مواد، دوربین‌های اضافی و غیره.
- عدم پشتیبانی از نهادهای طبیعی صاف که با نهادهای ۳ بعدی در یک ترسیم مخلوط شده‌اند.
- عدم پشتیبانی از ساخت نهادهای مانند نهادهای متقاطع، نهادهای متحد و غیره.
