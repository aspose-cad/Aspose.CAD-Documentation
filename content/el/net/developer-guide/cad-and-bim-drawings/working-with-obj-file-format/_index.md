Η Aspose.CAD υποστηρίζει τώρα τη μορφή αρχείου OBJ. Η μορφή αρχείου OBJ είναι μια 3D γεωμετρία που περιέχει χάρτες υφής, 3D συντεταγμένες, πολυγωνικές επιφάνειες και άλλες πληροφορίες αντικειμένων.

{{% /alert %}}
