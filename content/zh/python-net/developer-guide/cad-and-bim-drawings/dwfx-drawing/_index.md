## **DWFx 绘图**
DWFx 是 DWF 文件格式的最新版本。DWFx 文件是使用 Autodesk CAD 软件创建的 2D 或 3D 绘图。它基于 XML Paper Specification (XPS)，因此可以使用 Microsoft XPS 观察器查看和打印。
