## **DWFx 图纸**
DWFx 是 DWF 文件格式的最新版本。DWFx 文件是使用 Autodesk CAD 软件创建的 2D 或 3D 图纸。它基于 XML 纸张规范 (XPS)，因此可以使用 Microsoft XPS 查看器查看和打印。
