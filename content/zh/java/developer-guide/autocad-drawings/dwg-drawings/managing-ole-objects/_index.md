{{% alert color="primary" %}} 

对象链接和嵌入（OLE）是一个Microsoft Windows特征，它允许您在保持编辑原始应用程序中的信息能力的同时，将信息从一个应用程序复制或移动到另一个应用程序。基本上，OLE将来自不同应用程序的数据组合到一个文档中。要使用OLE，您需要支持OLE的源应用程序和目标应用程序。

{{% /alert %}}
