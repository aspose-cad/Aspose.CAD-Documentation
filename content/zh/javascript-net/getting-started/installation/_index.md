## **通过 npm 安装 Aspose.CAD for JavaScript**

要在您的应用程序中使用 Aspose.CAD for JavaScript，请在您的项目中从 [NPM](https://www.npmjs.com/@aspose-cad/) 安装 aspose.cad。之后，您可以以编程方式访问导出器。

**JavaScript**

{{< highlight python >}}
npm install aspose-cad
{{< /highlight >}}
