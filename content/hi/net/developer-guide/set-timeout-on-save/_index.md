## **सहेजने पर टाइमआउट सेट करें**

Aspose.CAD for .NET API आपको सहेजने पर एक टाइमआउट सेट करने की अनुमति देता है। यह उन मामलों में मददगार हो सकता है जहाँ सहेजने की प्रक्रिया बहुत समय ले रही है या बहुत अधिक मेमोरी का उपयोग कर रही है। इसके लिए, API  [**InterruptionTokenSource**](https://reference.aspose.com/cad/net/aspose.cad/interruptiontokensource) क्लास प्रदान करता है। [**InterruptionTokenSource**](https://reference.aspose.com/cad/net/aspose.cad/interruptiontokensource) क्लास एक टोकन प्रदान करती है जिसका उपयोग लंबे संचालन को बाधित करने के लिए किया जाता है।

## नमूना कोड

निम्नलिखित कोड स्निपेट [**InterruptionTokenSource**](https://reference.aspose.com/cad/net/aspose.cad/interruptiontokensource) क्लास के उपयोग को प्रदर्शित करता है।
