## **DWFx Малюнок**
DWFx є найновішою версією формату файлів DWF. Файл DWFx є 2D або 3D малюнком, створеним за допомогою програмного забезпечення Autodesk CAD. Він базується на Специфікації паперу XML (XPS) і, отже, може бути переглянутий і надрукований за допомогою Microsoft XPS Viewer.

Aspose.CAD для Java дозволяє розробникам відкривати та експортувати файли DWFx у формат PDF. Наступний код демонструє, як відкривати файли DWFx і зберігати їх у форматі PDF.
