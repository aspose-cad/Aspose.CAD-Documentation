## **Como converter DWG para PDF com Coordenadas**

Aspose.CAD fornece a funcionalidade de carregar o arquivo AutoCAD [DWG](https://docs.fileformat.com/cad/dwg/) e exportá-lo para o formato [PDF](https://docs.fileformat.com/pdf/) fornecendo coordenadas. Para isso, a API fornece a classe [**CadVportTableObject**](https://reference.aspose.com/cad/java/com.aspose.cad.fileformats.cad.cadtables/CadVportTableObject). A classe [**CadVportTableObject**](https://reference.aspose.com/cad/java/com.aspose.cad.fileformats.cad.cadtables/CadVportTableObject) é usada para especificar as coordenadas para conversão.
