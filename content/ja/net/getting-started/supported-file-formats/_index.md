## **まだサポートされていない**

- シャドウ、ライト、マテリアル、追加カメラなど。
- 同じ図面内での3Dエンティティと自然に平坦なエンティティの混在には対応していません。
- 交差エンティティ、統合エンティティなどの構築エンティティには対応していません。
