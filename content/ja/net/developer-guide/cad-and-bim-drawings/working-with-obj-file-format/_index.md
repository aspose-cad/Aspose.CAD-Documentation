Aspose.CADは現在OBJファイルフォーマットをサポートしています。OBJファイルフォーマットは、テクスチャマップ、3D座標、ポリゴン面、およびその他のオブジェクト情報を含む3Dジオメトリです。

{{% /alert %}}
