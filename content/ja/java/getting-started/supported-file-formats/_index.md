## **まだサポートされていないもの**

- シャドウ、ライト、マテリアル、追加カメラなど。
- 同じ図面内で 3D エンティティと混合された自然に平坦なエンティティのサポートがありません。
- 交差されたエンティティ、統合されたエンティティなどのようなエンティティのビルディングに対するサポートがありません。
