オブジェクトリンクおよび埋め込み（OLE）は、情報を一つのアプリケーションから別のアプリケーションにコピーまたは移動し、元のアプリケーションで情報を編集する能力を保持できるMicrosoft Windowsの機能です。基本的に、OLEは異なるアプリケーションからのデータを一つの文書に結合します。OLEを使用するには、OLEをサポートするソースとデスティネーションの両方のアプリケーションが必要です。

{{% /alert %}}
