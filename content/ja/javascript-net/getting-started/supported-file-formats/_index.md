## **まだサポートされていないもの**

- シャドウ、ライト、マテリアル、追加のカメラなど。
- 同じ描画内の自然に平坦なエンティティと 3D エンティティの混在には対応していません。
- 交差したエンティティ、一体化したエンティティなどの構築には対応していません。
