{{% alert color="primary" %}}

Aspose.CAD for JasperReportsをシステムにインストールするには、システムが以下の要件を満たす必要があります：

- Java 2 Platform Standard Edition 1.6 以上。
- JasperReports 6.3.0 以上。
- JasperReports Server 6.3.0 以上。

{{% /alert %}}
