Poniższy fragment kodu demonstruje dodawanie niestandardowych właściwości do pliku DXF za pomocą metody [**CadImage.Header.CustomProperties.Add**](https://reference.aspose.com/cad/java/com.aspose.cad.fileformats.cad.cadobjects/CadHeader#getCustomProperties--).
