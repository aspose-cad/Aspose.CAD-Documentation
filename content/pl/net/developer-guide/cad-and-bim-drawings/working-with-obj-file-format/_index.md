{{% alert color="primary" %}}

Aspose.CAD teraz obsługuje format plików OBJ. Format plików OBJ to geometria 3D, która zawiera mapy tekstur, współrzędne 3D, twarze wielokątne i inne informacje o obiektach.

{{% /alert %}}
