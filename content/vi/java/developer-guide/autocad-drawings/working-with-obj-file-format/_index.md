Aspose.CAD hiện hỗ trợ định dạng tệp OBJ. Định dạng tệp OBJ là một hình học 3D chứa các bản đồ kết cấu, tọa độ 3D, mặt đa giác, và thông tin về các đối tượng khác.

{{% /alert %}}
