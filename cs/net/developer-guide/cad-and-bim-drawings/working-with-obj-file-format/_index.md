{{% alert color="primary" %}}

Aspose.CAD nyní podporuje formát souboru OBJ. Formát souboru OBJ je 3D geometrie obsahující texturované mapy, 3D souřadnice, polygonální plochy a další informace o objektu.

{{% /alert %}}
