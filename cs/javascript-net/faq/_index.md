## **Obecné**
**Otázka: Co je to Aspose.CAD?**

Aspose.CAD pro JavaScript je knihovna tříd, která umožňuje vašim aplikacím provádět širokou škálu úloh zpracování dokumentů. S Aspose.CAD pro JavaScript můžete generovat, upravovat, konvertovat, renderovat a tisknout dokumenty bez použití aplikací třetích stran.

Podrobnosti naleznete v naší dokumentaci.

**Otázka: Proč Aspose.CAD?**

Proč uživatelé volí Aspose.CAD pro práci soubory? Nejkratší odpověď, kterou můžeme dát, je, že Aspose.CAD obsahuje sadu funkcí pro konverzi a vodoznakování a pokrývá širokou škálu úkolů aplikovatelných na soubory ve různých podporovaných formátech. Aspose poskytuje neomezenou bezplatnou technickou podporu pro všechny své produkty. Podpora je k dispozici všem uživatelům, včetně hodnocení. Hlavní oblast podpory je na [Aspose.Forums](https://forum.aspose.com/c/cad/19).

**Otázka: Co znamená „Aspose.CAD nevyžaduje aplikace třetích stran“?**

Aspose.CAD pro JavaScript nepotřebuje k práci instalaci žádných aplikací třetích stran ani další softwarové balíky. Stačí stáhnout a nainstalovat Aspose.CAD, jak je popsáno v článku ["Instalace"](cs/cad/javascript-net/installation/), a začít pracovat.

**Otázka: Jaké platformy podporuje Aspose.CAD?**

Aspose.CAD pro JavaScript pokrývá většinu populárních vývojových prostředí a nasazovacích platforem. Jeho API lze použít k vývoji aplikací pro širokou škálu operačních systémů, jako jsou Windows, Linux a Mac OS, a různé platformy.

Podrobnosti naleznete na stránce ["Přehled produktu"](cs/cad/javascript-net/product-overview/).

**Otázka: Jaké formáty souborů podporuje Aspose.CAD?**

Aspose.CAD pro JavaScript je samostatné API pro zpracování souborů CAD a BIM vhodné pro Angular, React, Vue, JavaScript, TypeScript. Poskytuje funkce pro konverzi formátů DXF, DWG, DGN, DWF, DWFX, IFC, STL, DWT, IGES, PLT, CF2, OBJ, HPGL, IGS, PCL, FBX, SVG do souborů ve vysoké kvalitě PDF a dalších vektorových a rastrových obrázkových souborů jako jsou: BMP, GIF, JPG, JPEG, JP2, PNG, TIFF, PSD, DICOM, WEBP, WMF, EMF.

Úplný seznam podporovaných formátů naleznete na stránce ["Podporované formáty souborů"](cs/cad/javascript-net/supported-file-formats/) nebo ["Konverze formátů souborů"](cs/cad/javascript-net/converting-file-formats/).

**Otázka: Jak získám [dočasnou licenci](https://purchase.aspose.com/temporary-license/)?**

Dočasná licence je časově omezená plná licence, která vám umožňuje vyzkoušet každý aspekt produktu před jeho zakoupením.

Dočasnou licenci můžete požádat tím, že přidáte produkty do košíku ve vedoucím průvodci nákupem. Alternativně můžete požádat o dočasnou licenci z existujícího návrhu.

Jakmile jste přidali produkty, které chcete otestovat, do košíku, pokračujte a na další stránce můžete požádat o dočasnou licenci.
