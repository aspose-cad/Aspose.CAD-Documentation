## **Update Hyperlink in a DWG**

Aspose.CAD for Java allows you to access the hyperlink in an AutoCAD drawing using the [**CadBaseEntity.getHyperlink()**](https://reference.aspose.com/cad/java/com.aspose.cad.fileformats.cad.cadobjects/CadBaseEntity#getHyperlink--). You can access each hyperlink from this collection one by one and edit its properties.
